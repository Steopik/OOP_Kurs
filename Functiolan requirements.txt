Данный проект представляет собой "рограммное средство для чтения книг с системой рекомендаций.
Стек технологий: C# + MAUI(.NET 8) + MySQL
Функционал:
1) Регистрация пользователя
2) Авторизация пользователя 
3) Просмотр списка рекомендованных книг, полученного на основании уже прочитанных с примененем искусственного интеллекта.
4) Поиск книг по названию
5) Фильтрация найденных книг по длине, автору и жанрам
6) Закачка выбранной книги
7) Удаление книги с устройства 
8) Просмотр книги в онлайн-режиме
9) Открытие книги со страницы, где была остановка


1) Регистрация пользователя - функция, которая позволяет пользователю создать новый аккаунт. Для этого программа запросит email, логин и пароль. Потом будет проверено наличие пользователя с таким логином. В случает наличия такого пользователя в регистрации будет отказано. После пароль будет проверен на допустимость. Потом будет отправлен код подтверждения на email. После ввода кода аккаунт будет создан.   
2) Авторизация пользователя - функция, котоая позволяет пользователю зайти в свой аккаунт. Необходимо будет ввести логин и пароль. После этого данные будут захешированы. Хеш будет отправлен в бд для подтвержденя. Если бд подтвердит правильность хеша, пользователь получит токен доступа.
3)  Просмотр списка рекомендованных книг - функция, которая будет рекомендовать книги, котоые предложит ии на основании ранее прочитанных книг.
4) Поиск книг по названию - функция, которая позволяет искать книги, в названиях которых содежится строка поиска введенная пользователем.
5) Фильтрация книг - это функция, которая позволяет фильтровать книги по автору, длине и жанрам.
6) Закача выбранной книги - это функция, которая поволяет скачать книгу на устройство и после использовать её без достуа в интеренет.
7) Удаление книги - функция для удаления с утсройства ранее скачанной книги.
8) Просмотр книги в онлайн-режиме - это функци, которая вначале будет загружать в память программы несколько страниц книги и подгружать их по мере перелстывания страниц.
9) Открытие книги со страницы, где была остановка - это функция, которая позваоляет открыть книгу на моменте остановки.
